□ 使用帮助

● 思维导图模块：

1. 点击「新建」创建思维导图

2. 粘贴讲义内容或上传文件

3. 复制提示词发给AI分析

4. 将AI返回的JSON粘贴回来

5. 保存并查看思维导图

● 范文库模块：

1. 点击「上传范文」添加范文

2. 填写标题、类型、内容、关键词

3. 可点击「AI分析提炼」自动提取重点

4. 保存后可搜索和高亮关键词

● 政策文件模块：

1. 点击「上传政策」添加政策文件

2. 粘贴政策原文

3. 点击「生成多版本」生成重点标注版/摘要版/挖空版

4. 保存后可切换不同版本查看

● 待办清单模块：

1. 输入任务内容，选择优先级

2. 点击「添加」创建任务

3. 点击任务前的方框标记完成

4. 使用筛选按钮查看不同状态的任务
